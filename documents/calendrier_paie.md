# Calendrier et Informations de Paie
> Mots clés : quand suis-je payé, date de paiement, versement salaire, jour de paie, date de virement, paiement mensuel

## Date de versement du salaire
Le salaire est versé **le 28 de chaque mois**.
Si le 28 tombe un week-end ou un jour férié, le versement est effectué le dernier jour ouvré avant cette date.

### Délais bancaires
- Les virements sont émis le 27 au soir
- Le délai de crédit dépend de votre banque (généralement J+1, parfois J+2 pour certaines banques)
- En cas de non-réception au 30 du mois, contacter le service paie

## Paie non reçue — Procédure d'urgence
Si vous n'avez pas reçu votre salaire à la date prévue :

1. **Vérifier d'abord** : Confirmez vos coordonnées bancaires sur le portail RH
2. **Contacter le service paie** : paie@entreprise.fr ou poste interne 1234
3. **Délai de résolution** : Le service paie dispose de 48h pour identifier la cause
4. **Causes fréquentes** : Changement de RIB non pris en compte, erreur de saisie, incident bancaire
5. **Solution d'urgence** : En cas de blocage avéré, un virement express peut être effectué sous 24h sur justification

## Changement de coordonnées bancaires
Pour changer votre RIB :

1. Connectez-vous sur le portail RH
2. Allez dans "Mon Profil" puis "Coordonnées bancaires"
3. Saisissez votre nouveau RIB (IBAN + BIC)
4. Joignez un RIB original en PDF
5. Validez — la modification est prise en compte pour le mois suivant si faite avant le 15

## Accès aux bulletins de paie
- **Portail web** : http://portail-rh.entreprise.fr > "Mes Documents" > "Bulletins de paie"
- **Application mobile** : iOS et Android disponibles
- Les bulletins sont disponibles le 25 de chaque mois pour le mois en cours

## Erreur sur le bulletin de paie
Si vous constatez une erreur sur votre bulletin :

1. **Identifier l'erreur** : Notez précisément la ligne concernée et le montant attendu
2. **Ouvrir un ticket paie** : Via le portail RH > "Signaler une anomalie paie"
3. **Joindre les justificatifs** : Contrat, avenant, relevé de pointage selon le cas
4. **Délai de régularisation** : La correction est effectuée sur le bulletin du mois suivant
5. **Nouveau bulletin** : Un bulletin rectificatif est émis et archivé dans votre coffre-fort numérique
